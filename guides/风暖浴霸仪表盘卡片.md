# 风暖浴霸仪表盘卡片

本文档适用于通过 `bathroom_heater_0820_fv_rb20vl1` profile 接入的松下风暖浴霸 `FV-RB20VL1`。

该设备在 Home Assistant 中映射为 `climate` 实体，支持以下模式：

| Home Assistant 模式 | 浴霸功能 |
| --- | --- |
| `heat` | 取暖 |
| `fan_only` | 换气 |
| `cool` | 凉干燥 |
| `dry` | 热干燥 |
| `off` | 待机 |

## 原生卡片

Home Assistant 原生恒温器卡片可以直接切换浴霸模式：

```yaml
type: thermostat
entity: climate.your_bathroom_heater
name: 风暖浴霸
```

请将 `climate.your_bathroom_heater` 替换为实际实体 ID。

## Mushroom 五模式卡片

以下示例需要先安装 [Mushroom Cards](https://github.com/piitaya/lovelace-mushroom)。

```yaml
type: grid
columns: 5
square: false
cards:
  - type: custom:mushroom-template-card
    primary: 取暖
    icon: mdi:weather-sunny
    icon_color: >-
      {% if is_state('climate.your_bathroom_heater', 'heat') %}
        orange
      {% else %}
        grey
      {% endif %}
    layout: vertical
    tap_action:
      action: call-service
      service: climate.set_hvac_mode
      target:
        entity_id: climate.your_bathroom_heater
      data:
        hvac_mode: heat

  - type: custom:mushroom-template-card
    primary: 换气
    icon: mdi:fan
    icon_color: >-
      {% if is_state('climate.your_bathroom_heater', 'fan_only') %}
        green
      {% else %}
        grey
      {% endif %}
    layout: vertical
    tap_action:
      action: call-service
      service: climate.set_hvac_mode
      target:
        entity_id: climate.your_bathroom_heater
      data:
        hvac_mode: fan_only

  - type: custom:mushroom-template-card
    primary: 凉干燥
    icon: mdi:snowflake
    icon_color: >-
      {% if is_state('climate.your_bathroom_heater', 'cool') %}
        cyan
      {% else %}
        grey
      {% endif %}
    layout: vertical
    tap_action:
      action: call-service
      service: climate.set_hvac_mode
      target:
        entity_id: climate.your_bathroom_heater
      data:
        hvac_mode: cool

  - type: custom:mushroom-template-card
    primary: 热干燥
    icon: mdi:water-percent
    icon_color: >-
      {% if is_state('climate.your_bathroom_heater', 'dry') %}
        blue
      {% else %}
        grey
      {% endif %}
    layout: vertical
    tap_action:
      action: call-service
      service: climate.set_hvac_mode
      target:
        entity_id: climate.your_bathroom_heater
      data:
        hvac_mode: dry

  - type: custom:mushroom-template-card
    primary: 待机
    icon: mdi:power
    icon_color: >-
      {% if is_state('climate.your_bathroom_heater', 'off') %}
        red
      {% else %}
        grey
      {% endif %}
    layout: vertical
    tap_action:
      action: call-service
      service: climate.set_hvac_mode
      target:
        entity_id: climate.your_bathroom_heater
      data:
        hvac_mode: "off"
```

请将示例中的所有 `climate.your_bathroom_heater` 替换为实际实体 ID。
